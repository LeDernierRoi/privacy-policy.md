# Gizlilik Politikası

**Berkay İpekçi Tag Botu**

Bu bot, Discord sunucularında sunucu etiketi (tag) takip ve ödül sistemi sunmaktadır.

### Toplanan Veriler
- Kullanıcıların Discord ID’si
- Sunucu etiketi takma tarihi
- Etiket süresi (kaç gündür taktığı)

### Verilerin Kullanımı
- Veriler yalnızca tag takip sistemi ve otomatik rol verme amacıyla kullanılır.
- Veriler Discord dışında hiçbir yerde saklanmaz.
- Hiçbir üçüncü taraf ile paylaşılmaz.
- Makine öğrenimi veya yapay zeka eğitimi için kullanılmaz.

### Veri Silme
Kullanıcı sunucudan ayrıldığında veya sunucu etiketini çıkardığında ilgili tüm veriler otomatik olarak silinir.

### İletişim
Herhangi bir soru, öneri veya veri silme talebi için Discord üzerinden **orhannn** adlı kullanıcıya ulaşabilirsiniz.
